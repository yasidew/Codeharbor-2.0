Weighted Class Complexity (WCC) Guidelines for Java

Type of Control Structure and Weights:

  • Sequential: 0
  • Branch (if, else, else if): 1
  • Iterative (for, while, do-while): 2
  • Switch statement with n cases: n

Nesting Levels and Weights:

  • Sequential statements: 0
  • Statements inside the outermost level: 1
  • Statements inside the second level: 2
  • Statements inside the nth level: n

Inheritance Levels and Weights:

  • Statements inside the base class: 0
  • Statements inside the first derived class: 1
  • Statements inside the nth derived class: n

Try-Catch Block Weights:

  • Simple try-catch: 1
  • Try-catch-finally: 2
  • Nested try-catch with level n: n

Compound Conditional Weights:

  • Simple condition: 1
  • Compound condition with n logical operators: n

Thread Weights:

  • Simple thread creation: 2
  • Thread synchronization: 4

CBO Weights:

  • Loose coupling (e.g., dependency injection, interfaces): Weight 1
  • Tight coupling (e.g., direct field access, extensive inter-class dependencies): Weight 3

MPC Weights:

  • Simple message passing (e.g., single method call): Weight 1
  • Moderate message passing (e.g., multiple method calls, callback functions): Weight 2
  • Complex message passing (e.g., event-driven messaging, asynchronous calls): Weight 3
